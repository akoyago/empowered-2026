# Summarize the long stuff

**Use case:** Condense a long grant write-up, fund history, or donor profile into a quick brief.

**Prompt**
```
Summarize the record below for a [board update | quick internal brief].
Use only what is in the record — do not add or assume anything.

Return:
1. One-line status
2. Key dates
3. Amounts / decisions
4. Last touchpoint
5. Anything still open

Keep it under [120] words, plain language.

Record:
[paste record]
```

# Rewrite & reformat

**Use case:** Tighten tone, shorten, or restyle existing copy for a board packet or a donor email.

**Prompt**
```
Rewrite the text below for [a board packet | a major donor | a grantee].
Goals: [tighten | shorten to ~[X] words | warmer tone | plain language].

Keep all facts and figures unchanged — do not add new claims.
Return [the rewritten version only | a clean version plus a one-line note on what you changed].

Text:
[paste text]
```
